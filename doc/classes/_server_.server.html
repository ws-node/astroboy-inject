    Set a scoped service

    • service should be decorated by @Injectable(...)

    Set a scoped service with injectable token (such abstract class, but not the typescript interface because there's no interface in the javascript runtime) and a well-created implement service instance. All scoped services will be created new instance in different request pipe (but injecting by instance means the instance may be changed in runtime, so please be careful. If you want to prevent this situation, use a service factory here).
